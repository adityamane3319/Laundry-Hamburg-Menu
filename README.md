# Laundry Services Hero Section

This is a simple responsive web page for a laundry service.

## Features
- Navbar with logo, links, and username
- Hero section with text and image
- Responsive design for desktop, tablet, and mobile

## Technologies Used
- HTML
- CSS (Flexbox + Media Queries)

## Responsive Behavior
- Desktop: Two-column layout (text + image)
- Tablet: Stacked layout with resized text and image
- Mobile: Navigation links hidden, only logo and username shown
Mobile hamburger menu (CSS only)
